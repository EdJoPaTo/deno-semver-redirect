Deno Semver Redirect

This service allows you to use Deno dependencies via semantic versioning. See also this Deno Issue on GitHub.

This service works via redirects. The requested version range is parsed and the request is redirected to the newest fitting version on deno.land/x. This saves resources for this service and provides you with exactly the sources on deno.land/x. For more details check the README on GitHub.

Try the following redirects:
grammy@0.3
std@0.x
